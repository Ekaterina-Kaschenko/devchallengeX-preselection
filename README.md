Для запуску проекту необходно встановити залежності за допомогою команди
```npm install```

Після цього проект може бути запущено за допомогою команди
```npm run start:prod```

Стек технологій
- Node.js (express server)
- React (Redux, redux-saga, reselect)
- Immutable.js
- lodash
- Bootstrap
